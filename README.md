# Dairy Barn

This project done using XAML and razor pages interfaces. It is a design off of Dairy Queen that has a menu and other features that helped with learning the OO principles. Built as part of CIS 400 Object Oriented Design.

## Technologies
- C# / .NET
- XAML
- Razor Pages

## Features
- Allows for the user to customize menu items by clicking of the item.
- Has a running total that changes with each customization option on an item and when new items get added to the to order.
- Has a checkout feature that lets the user checkout their order and gives a receipt in a text file called "Recipts.txt" in the bin folder.
- A list of all possible options that a customer can order along with ways to sort and filter the results.
- Lets the user leave reviews about the restaurant

## How to Run
1. Clone the repository
2. Set the start up project
	- Option 1: PointOfSale is the XAML UI that lets the user interact with the menu
	- Option 2: Website is the Razor Pages app that lets the user see all available options and filter the results
	- Option 3: Can use a custom start up to start both up at the same time
3. Build and Run
